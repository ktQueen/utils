(static) toChinese(arabic) → {string}

阿拉伯数字转中文数字, 如果传入数字时则最多处理到21位，超过21位js会自动将数字表示成科学计数法，导致精度丢失和处理出错 传入数字字符串则没有限制

arabic number | string 阿拉伯数字
