(static) setCookie(name, value, timeopt, domainopt, pathopt)

设置cookie
Parameters:
Name Type Attributes Default Description
name string cookie名
value string cookie值
time string | number <optional>
 cookie的有效时间，如果不设置expres为session。关闭浏览器后或者过了session时间就清除cookie, time参数可以分两种情况： a. 如果是整数，则代码内部直接算绝对过期时间 b. 如果是2016-01-01 00:00:00这种格式则直接设置为过期时间
domain string <optional>
 cookie的域名，默认为js文件所在域名
path String <optional>
 '/' cookie路径，默认为当前路径js文件所在路径
